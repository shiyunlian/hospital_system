# hospital_system

This project is built with React, Node.js and Oracle database.

## Available Scripts

To run this project, clone the repository to your computer using `git clone https://github.com/shiyunlian/hospital_system.git`

In the project directory:

Run client side, execute `cd client` to get into the client directory, execute `npm i` to install the required dependencies, execute `npm start`

Run server side, execute `cd server` to get into the server directory, execute `npm i` to install the required dependencies, execute `npm start`
